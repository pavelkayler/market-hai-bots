# Local run (Momentum + Manual DEMO)

1. Backend
   - `cd backend`
   - `npm install`
   - `npm run dev`

2. Frontend
   - `cd frontend`
   - `npm install`
   - `npm run dev`

3. Open pages:
   - Momentum: `http://localhost:5173/momentum`
   - Universe Search: `http://localhost:5173/universe`
   - Manual DEMO: `http://localhost:5173/manual-demo`
